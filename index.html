PyBuilder, hooray!

An extendible, easy to use continuous build tool for Python. PyBuilder is a continuous build tool for multiple languages. It primarily targets Python projects but due to its extendible nature it can be used for other languages as well. PyBuilder features a powerful yet easy to use plugin mechanism which allows programmers to extend the tool in an unlimited way.

PyBuilder at github »

Installation

PyBuilder is easy to install using pip.

pip install pybuilder
