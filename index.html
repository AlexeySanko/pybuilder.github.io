PyBuilder

An easy to use continuous build tool for Python.

PyBuilder is primarily targeted at Python projects but due to its extensible nature it can be used for other languages as well. PyBuilder features a powerful yet easy to use plugin mechanism which allows programmers to extend the tool in an unlimited way.

Installation

Using pip PyBuilder is easy to install:

$ pip install pybuilder

If you use virtualenv we recommend installing pybuilder inside it so that it does not see your system packages.

Compatibility

PyBuilder supports a lot of tools typically used when building a Python application, such as unittest, coverage, pychecker, pylint and setuptools.

Runs on python 2.6 - 3.3

PyBuilder runs on Python 2.6 to 3.3 and PyPy. Every checkin is tested on versions 2.6, 2.7, 3.2, 3.3 and PyPy on

Get started

To get started with a simple Python project that has unittests create a file build.py with the following content: 

from pybuilder.core import use_plugin

use_plugin("python.core")
use_plugin("python.unittest")
use_plugin("python.coverage")
use_plugin("python.distutils")

default_task = "publish"

Create a directory src/main/python and put all your Python code there. Create a directory src/unittest/python and put all your unittest modules there, each named *_tests.py .

In the root directory, run pyb : 

$ pyb
