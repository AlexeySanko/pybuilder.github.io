Start a project 


#!/bin/sh
pip install pybuilder
pyb --start-project
pyb install_dependencies publish

Try it yourself »

Build an existing project 


    git clone https://github.com/aelgru/fluentmock
    cd fluentmock
    virtualenv venv; source venv/bin/activate
    pip install pybuilder
    pyb install_dependencies
    pyb

Install PyBuilder

PyBuilder is available on PyPI 

pip install pybuilder

We recommend using virtualenv . Please see the dedicated installation page .

Open Source

PyBuilder is licensed under Apache 2.0 License and hosted on github.com/pybuilder/pybuilder .

Compatible

PyBuilder supports many tools typically used when building a Python application, such as unittest, coverage, pychecker, pylint and setuptools.

Runs on Python 2 / 3

PyBuilder runs on Python 2.6 to 3.3 and PyPy . Every commit is tested on versions 2.6, 2.7, 3.2, 3.3 and PyPy on Travis-CI.
